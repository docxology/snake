# Snake-in-the-Box Documentation

Welcome to the comprehensive documentation for the Snake-in-the-Box Python package. This package implements the heuristically-pruned breadth-first search algorithm from Ace (2025) for finding long induced paths in n-dimensional hypercube graphs.
